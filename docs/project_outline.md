# 项目大纲

## 项目名称

基于图像识别的种子发芽率自动统计与活力评估系统

英文：SeedGerm-Vigor: Image-based Seed Germination Counting and Vigor Assessment System

## 项目定位

面向种子质检与发芽实验场景，利用培养皿图像自动检测种子、判断发芽状态、重建发芽曲线，并生成活力评估报告。

## 核心问题

传统发芽实验依赖人工定时观察和计数，存在耗时、主观性强、统计记录不连续等问题。公共时序图像数据集为自动化统计提供了可训练、可验证的数据基础。

## 系统流程

```text
培养皿图像/时序图像
→ 种子目标检测
→ 发芽/未发芽状态识别
→ 发芽率统计
→ 发芽曲线重建
→ T50、平均发芽时间、发芽速度指数计算
→ 活力评分与质检报告
```

## 创新点

1. 从单次人工计数转向时序自动统计。
2. 从简单发芽率扩展到 T50、平均发芽时间、发芽速度和整齐度等多指标活力评估。
3. 将检测结果聚合为批次级质检报告，而不只是单张图像分类。
4. 支持后续接入公共数据集训练模型，形成可验证实验链条。
